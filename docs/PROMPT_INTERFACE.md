# Prompt Interface: EmberHeart
Version: 1.0

## Output Structure (Mandatory Order)

### 1) TIME
Week index, season, and notable events.

### 2) DASHBOARD
- Food: [Status] (FU stock)
- Fuel: [Status] (HU stock)
- Structural: [Status]
- Morale: [-10..+10]
- Tremor: [0..10]
- Legitimacy: [0..100]
- Faction Pressure: Sparklines or numbers for Concordat/Valemere/etc.

### 3) TOP CHANGES
Max 5 bullets explaining current tick shifts.

### 4) ACTIVE THREATS
Max 3 threats with severity and horizon.

### 5) DECISION GATE (If Triggered)
- Tier Level (2-4).
- Context and Options (A, B, C...).
- Projected Impacts (One-liners).
