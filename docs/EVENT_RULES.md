# Event Rules: EmberHeart
Version: 1.0

## 每周事件掷骰 (Weekly Event Roll)
- **Base Chance**: 40%
- **Modifiers**:
    - Low Food: +10%
    - Critical Food: +20%
    - Low Morale (<= -4): +10%
    - High Tremor (>= 5): +10%
    - Failing Integrity: +15%

## 事件分类 (Categories)
- **Minor (60%)**: Livestock illness, rumors, minor injuries.
- **Moderate (30%)**: Tunnel collapse, scout sightings, theft.
- **Major (10%)**: Deep creature emergence, Faction ultimatum (Requires Tier 3).

## ⚡ Immediate Feedback Events (IFE)
*Triggered by specific player actions between weekly ticks.*
- **Resource Salvage**: Small DC check for +2-5 FU/WU/OU for immediate scavenging.
- **NPC Insight**: Single NPC loyalty +2 for a successful roleplay interaction.
- **Tremor Ripple**: Fear +1 settlement-wide if a player uses high-impact explosives/spells.

## 特殊触发器 (Campaign Triggers)
- **Open Call Posted**: Eligible if Tremor Threat >= 3 or severe resource loss.
- **Adventurer Arrival**: Tier 3 prompt for user to select arrivals.

## 政治影响 (Legitimacy Impact)
- Resolve Moderate Threat: +3 to +6.
- Resolve Major Threat: +8 to +15.
- Civilian Death: -8.
- Settlement Collapse: -10.
