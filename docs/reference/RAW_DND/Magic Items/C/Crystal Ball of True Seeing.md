# [Crystal Ball of True Seeing](https://www.dndbeyond.com/magic-items/4864-crystal-ball-of-true-seeing)

#Wondrous *Item*, #Legendary *(Requires attunement)*

While touching this crystal orb, you can cast [[Scrying]] (save DC 17) with it. In addition, you have [Truesight](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Truesight) with a range of 120 feet centered on the spell’s sensor.
