# [Dusk Dagger](https://www.dndbeyond.com/magic-items/9445107-dusk-dagger)

#Weapon *(dagger),* *rarity varies*

This delicately curved dagger is ornamented with several crescent moon designs. When you hit a creature who is completely in dim light or darkness with this magic weapon, that creature takes an extra 1d4 ( #Uncommon ), 1d8 ( #Rare ), or 1d12 ( #VeryRare ) Psychic damage.
