# [Veteran’s Cane](https://www.dndbeyond.com/magic-items/9308690-veterans-cane)

#Wondrous *Item*, #Common

As a Bonus Action, you can transform this walking cane into an ordinary [Longsword](https://www.dndbeyond.com/equipment/4-longsword) or change the Longsword back into a walking cane. In either case, you must be holding the item.
