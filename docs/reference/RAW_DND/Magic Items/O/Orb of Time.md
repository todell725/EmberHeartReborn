# [Orb of Time](https://www.dndbeyond.com/magic-items/9228913-orb-of-time)

#Wondrous *Item*, #Common

This orb can be used as an Arcane Focus.

While holding the orb, you can take a [Magic](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#MagicAction) action to determine whether it is morning, afternoon, evening, or nighttime. This property functions only on the [[Material Plane]].
