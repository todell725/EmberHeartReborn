# [Potion of Clairvoyance](https://www.dndbeyond.com/magic-items/4701-potion-of-clairvoyance)

#Potion, #Rare

When you drink this potion, you gain the effect of the [[Clairvoyance]] spell (no [Concentration](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Concentration) required).

An eyeball bobs in this potion’s yellowish liquid but vanishes when the potion is opened.
