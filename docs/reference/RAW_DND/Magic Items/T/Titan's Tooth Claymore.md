# [Titan’s Tooth Claymore](https://www.dndbeyond.com/magic-items/9445547-titans-tooth-claymore)

#Weapon *(greatsword),* #VeryRare 

This massive two-handed blade is made from the lost or harvested tooth of a [tarrasque](https://www.dndbeyond.com/monsters/5195232-tarrasque). You gain a +2 bonus to attack and damage rolls made with this magic weapon. If it hits an object, the hit is automatically a critical hit. In addition, if you roll a 20 on an attack roll using this weapon, the target takes an extra 2d6 slashing damage.
