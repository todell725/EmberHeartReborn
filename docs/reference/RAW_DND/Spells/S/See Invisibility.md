# [See Invisibility](https://www.dndbeyond.com/spells/2619011-see-invisibility)

#Level2 #Divination ( #Bard, #Sorcerer, #Wizard)

**Casting Time:** Action

**Range:** Self

**Components:** V, S, M (a pinch of talc)

**Duration:** 1 hour

For the duration, you see creatures and objects that have the [Invisible](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#InvisibleCondition) condition as if they were visible, and you can see into the [[Ethereal Plane]]. Creatures and objects there appear ghostly.
