# [True Seeing](https://www.dndbeyond.com/spells/2619201-true-seeing)

#Level6 #Divination ( #Bard, #Cleric, #Sorcerer, #Warlock, #Wizard)

**Casting Time:** Action

**Range:** Touch

**Components:** V, S, M (mushroom powder worth 25+ GP, which the spell consumes)

**Duration:** 1 hour

For the duration, the willing creature you touch has [Truesight](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Truesight) with a range of 120 feet.
