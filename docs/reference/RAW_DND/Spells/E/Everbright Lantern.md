# [Everbright Lantern](https://www.dndbeyond.com/magic-items/215549-everbright-lantern)

#Wondrous item, #Common 

This bullseye lantern contains an Eberron dragonshard that sheds light comparable to that produced by a [[Continual Flame]] spell. An _everbright lantern_ sheds light in a 120-foot cone; the closest 60 feet is bright light, and the farthest 60 feet is dim light.
