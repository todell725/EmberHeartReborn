# [Darkvision](https://www.dndbeyond.com/spells/2619082-darkvision)

#Level2 #Transmutation ( #Druid, #Ranger, #Sorcerer, #Wizard)

**Casting Time:** Action

**Range:** Touch

**Components:** V, S, M (a dried carrot)

**Duration:** 8 hours

For the duration, a willing creature you touch has [Darkvision](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Darkvision) with a range of 150 feet.
