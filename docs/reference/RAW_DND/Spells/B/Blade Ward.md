# [Blade Ward](https://www.dndbeyond.com/spells/2618931-blade-ward)

#Abjuration #Cantrip ( #Bard, #Sorcerer, #Warlock, #Wizard)
**Casting Time:** Action

**Range:** Self

**Components:** V, S

**Duration:** [Concentration](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Concentration), up to 1 minute

Whenever a creature makes an attack roll against you before the spell ends, the attacker subtracts 1d4 from the attack roll.
