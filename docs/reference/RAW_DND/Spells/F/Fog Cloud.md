# [Fog Cloud](https://www.dndbeyond.com/spells/2618910-fog-cloud)

#Level1 #Conjuration ( #Druid, #Ranger, #Sorcerer, #Wizard)

**Casting Time:** Action

**Range:** 120 feet

**Components:** V, S

**Duration:** [Concentration](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#Concentration), up to 1 hour

You create a 20-foot-radius [Sphere](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#SphereAreaofEffect) of fog centered on a point within range. The Sphere is [Heavily Obscured](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#HeavilyObscured). It lasts for the duration or until a strong wind (such as one created by [[Gust of Wind]]) disperses it.

**_Using a Higher-Level Spell Slot._** The fog’s radius increases by 20 feet for each spell slot level above 1.
